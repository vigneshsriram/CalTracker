README:

Steps to run the project:

1. Install MongoDb and go to administrator cmd prompt and type "net start mongodb" to start Mongob.
2. Open Nodejs Cmd prompt.
3. Go to the directory in which the project is saved.
4. Type "npm run seed" to seed the database.
5. Once done seeding, Type "npm start" to start the server.
6. Open your Browser and type "localhost:3000" to run the website.
7. Navigate your way through the website.
